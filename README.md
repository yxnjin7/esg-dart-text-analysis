# DART 사업보고서 기반 ESG 공시 텍스트 분석

## 1. 프로젝트 개요 & 동기

국내 기업의 DART 사업보고서에 포함된 ESG 관련 서술과 실제 ESG 평가 등급 간 관계 분석 프로젝트

단순 ESG 단어 빈도 확인이 아니라, 사업보고서의 ESG 언어가 실제 ESG 수준을 반영하는지 또는 기업 규모와 공시 분량의 영향을 받는지 검증

**핵심 질문**

> 사업보고서의 ESG 텍스트는 실제 ESG 수준의 신호인가,
> 아니면 기업 규모와 공시 역량의 반영인가?

## 2. 데이터 출처 & 전처리 과정

### 데이터 출처

* DART 사업보고서
* KCGS ESG 평가 등급
* 기업 기본 정보 및 통제 변수

### 분석 대상

* 127개 기업
* 3개 회계연도
* 총 381개 기업-연도 데이터

### 전처리 과정

1. OpenDART API 기반 기업별 사업보고서 수집
2. `stock_code → corp_code → rcept_no → fiscal_year` 기준 보고서 식별
3. 사업보고서 핵심 섹션 추출

   * II. 사업의 내용
   * IV. 이사의 경영진단 및 분석의견
   * VI. 이사회 등 회사의 기관
4. Kiwi 형태소 분석기 기반 텍스트 토큰화
5. 명사, 동사, 형용사, 영문 표현 중심 토큰 정제
6. 기업-연도 단위 분석 패널 구성

## 3. 분석 방법 & 선택 이유

ESG 공시 텍스트를 단일 방식으로 측정할 경우 특정 표현 방식에 결과가 의존할 가능성 존재
이를 보완하기 위한 네 가지 텍스트 점수 비교

| 분석 방법              | 설명                                    |
| ------------------ | ------------------------------------- |
| Seed TF-IDF        | E/S/G별 핵심 ESG seed 단어 기반 TF-IDF 점수 산출 |
| FastText 확장 TF-IDF | FastText 유사 단어를 활용한 ESG 사전 확장 및 점수화   |
| Dense Cosine       | 문장 임베딩 기반 ESG 의미 유사도 측정               |
| Dense Supervised   | ESG 등급을 학습한 supervised 방식의 텍스트 점수 산출  |

### 통계 분석

* Spearman 상관분석
* Mann-Whitney 검정
* 순서형 로짓 회귀분석
* OLS 회귀분석
* 이진 로짓 분석
* 기업 규모, 공시 분량, 업종, 연도 통제 후 추가 설명력 검토

## 4. 핵심 결과

### 1) 통제 전 ESG 텍스트 점수와 ESG 등급 간 양의 관계

Seed TF-IDF 기준 E/S/G 텍스트 점수와 ESG 등급 간 양의 상관관계 확인
특히 G 점수와 G 등급의 Spearman 상관계수 0.510

### 2) 기업 규모 효과의 강한 영향

기업 규모 변수만으로 통합 ESG 등급 분산의 약 43.3% 설명
공시 분량, 업종, 연도 추가 시 설명력 약 50.7%까지 증가

ESG 텍스트 점수와 등급 간 관계가 순수한 ESG 서술의 질뿐 아니라, 기업 규모와 공시 역량의 영향을 함께 반영한다는 점 확인

### 3) 텍스트 점수의 제한적인 추가 설명력

기업 규모, 공시 분량, 업종, 연도 통제 후 텍스트 점수 추가 시 설명력 증가 폭 약 0.7%p~5.3%p 수준

ESG 텍스트가 등급과 관련은 있으나, 규모 효과를 넘어서는 독립적 설명력은 제한적이라는 해석

### 4) 완전통제 후 일부 지배구조 신호만 잔존

완전통제 후 Seed TF-IDF의 G 점수만 비교적 강건한 신호 유지

이사회, 사외이사 등 반복적 표현보다 윤리, 준법, 컴플라이언스, 부패방지 등 선택적 지배구조 표현의 중요성 확인

## 5. 시각화

* ESG 등급 분포
* 등급별 사업보고서 핵심 섹션 길이 분포
* 텍스트 점수와 ESG 등급 간 상관관계
* 통제 변수 추가에 따른 설명력 변화
* 표현 방식별 E/S/G 신호 비교

시각화를 통해 단순한 “긴 보고서 = 높은 ESG 등급” 해석의 한계 확인
기업 규모와 공시 분량을 함께 고려한 해석 필요성 도출

## 6. 한계점 & 향후 개선 방향

### 한계점

* KCGS ESG 등급 중심의 단일 평가 기준
* 127개 기업, 3개년으로 제한된 분석 범위
* 사업보고서 텍스트와 실제 ESG 성과 간 구분의 어려움
* Dense Embedding 방식에서 E/S/G 차원 구분의 한계
* 공시 충실도 요인과 ESG 성과 신호의 혼재 가능성

### 향후 개선 방향

* 복수 ESG 평가기관 등급 비교
* 기업 규모별, 업종별 차이 분석
* ESG 공시 의무화 전후 비교
* 수치 정보, 목표, 실행 계획 여부를 반영한 지표 추가
* 문장 단위 의미 분석 고도화

## 7. 배운 점

비정형 텍스트 데이터를 정량화할 때 단어 빈도만으로 결과를 해석하는 방식의 한계를 확읺하였다.

또한, ESG 공시 텍스트 분석에서 텍스트의 양, 기업 규모, 업종 특성, 평가기관 기준을 함께 고려해야 할 필요성을 도출하였고
통제 변수 없이 단순 상관관계만으로 결론을 내릴 경우 잘못된 해석으로 이어질 수 있다는 점을 확인하였다.

Seed TF-IDF, FastText, Dense Embedding 등 측정 방식에 따라 결과가 달라질 수 있으므로, 여러 방법 비교와 강건성 검토가 중요하다.
